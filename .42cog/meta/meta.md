# 项目元信息 (Meta)

## 基本信息

**项目名称**: DevForge

**项目标语**: 用代码锻造未来，以AI驱动表达

**版本**: 0.1.0

**创建日期**: 2026-02-18

**负责人**: Yeebin Huang

## 项目描述

DevForge 是一个面向大四计算机专业学生的个人技术展示网站，集项目作品集、技术博客、在线简历于一体。它深度集成 AI 能力，提供智能项目推荐、博客摘要生成等增强体验，以现代化的设计语言和流畅的交互展现开发者的技术深度与工程素养，助力求职与个人品牌建设。

## 目标用户

- 招聘方/HR: 通过结构化的项目展示和技术博客快速评估候选人的技术能力与工程经验
- 技术同行: 浏览技术文章、了解项目实现细节，进行技术交流与合作
- 开源社区: 发现可复用的项目模板和技术方案，参与讨论和贡献

## 核心价值主张

1. 技术深度展示 — 通过项目作品集、技术博客、代码片段等多维度内容，立体呈现开发者的技术栈广度与工程实践深度
2. AI增强体验 — 集成 AI 驱动的项目智能推荐、博客自动摘要、交互式技术问答等功能，让访客获得个性化的浏览体验
3. 现代化设计 — 基于 Next.js 15 + Tailwind CSS + shadcn/ui 构建，响应式布局、暗色模式、流畅动画，体现前沿的前端工程能力

## 技术栈

**前端**:
- 框架: Next.js 15
- UI: Tailwind CSS + shadcn/ui
- 语言: TypeScript

**后端**:
- 运行时: Bun / Node.js
- 数据库: PostgreSQL
- ORM: Drizzle ORM

**基础设施**:
- 托管: Vercel / EdgeOne Pages
- 数据库: Neon / Supabase
- 认证: Better Auth

**AI集成**:
- AI SDK: Vercel AI SDK
- 模型（优先级）: DeepSeek (deepseek-chat) > Kimi (moonshot-v1-auto) > Qwen (qwen-plus) > MiniMax (abab6.5s-chat)
- Fallback: Claude / GPT-4
